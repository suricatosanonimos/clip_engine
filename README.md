O sistema Г© dividido em duas partes independentes:

- **`/app`** вҖ” App Flet em Python. Roda no Android e Desktop. Toda a UI, configuraГ§Гөes e download de clipes. Comunica com a API via `httpx`.

## вҡЎ Pipeline de Processamento

![Pipeline de Processamento](docs/pipeline.svg)

1. **Download** вҖ” `yt-dlp` baixa o vГӯdeo do YouTube com retry automГЎtico (3x). Ou recebe upload direto do app.
2. **Tracking** вҖ” `MediaPipe` detecta os rostos frame a frame e aplica crop dinГўmico no formato 9:16 com `OpenCV`.
3. **Legendas** вҖ” `faster-whisper` transcreve o ГЎudio offline e `FFmpeg` queima as legendas com CRF 23.
4. **Upload** вҖ” `httpx` faz upload direto para o Supabase Storage (bucket `clips`) com timeout de 10 minutos.
5. **Entrega** вҖ” Signed URL de 7 dias Г© gerada e enviada ao app via SSE. Download disponГӯvel na Galeria.

### 1. Clone o repositГіrio


### 2. Configure a API

```bash

```

Crie o arquivo `.env` na pasta `api/`:

```env

```

### 3. Configure o App

```bash

```

### 4. Crie os buckets no Supabase

No painel do Supabase вҶ’ Storage вҶ’ New bucket:
- `clips` (privado)
- `videos` (privado)

### 5. Suba tudo

```bash

# Terminal 2 вҖ” ngrok (domГӯnio fixo gratuito)
```
